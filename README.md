## ▶️ Como executar

### Pré-requisito

- [Python 3](https://www.python.org/downloads/) instalado

### Executando um exercício

```bash
python secao-01-logica/EX01.py
python secao-04-selecao/EX30.py
```

> Nenhum pacote externo necessário — apenas Python puro (`math` e `datetime` são módulos da biblioteca padrão).
